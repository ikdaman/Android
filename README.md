## 읽다만(Ikdaman) Android App

### 개발 환경
- sdk : 29 ~ 35
- Kotlin : 2.1.0
- 자세한 정보는: [libs.versions.toml](https://github.com/ikdaman/Android/blob/master/gradle/libs.versions.toml) 참조

### 모듈
- app : Android App
- build-logic : 빌드 로직 모듈
- core : 공통 모듈
- feature : 기능 모듈
- main : Main 모듈
